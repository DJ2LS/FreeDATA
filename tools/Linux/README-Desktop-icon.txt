Copy the file "FreeDATA.Desktop" to the "Desktop" directory under your home directory.

cp FreeDATA.Desktop ~/Desktop

Edit ~/Desktop/FreeDATA.Desktop and change "USER" in the Icon line 
to your login account name and adjust the path to the FreeDATA icon as needed.

If you are running on as Raspberry Pi, open the File Browser (the yellow "Folder" icon at the top of the screen).

Go to Edit, Preferences.

Click the box next to "Don't ask options on launch executable file".

Click "Close" in the bottom right.

Logout then log back in, and then you should be able to double click the "FreeDATA" icon to run the program.
